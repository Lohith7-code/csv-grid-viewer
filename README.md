# CSV Grid Viewer (WinForms)

A simple Windows Forms application that loads CSV files into a read-only
DataGridView and allows viewing individual cell values in a popup dialog.

## Features
- Load CSV files dynamically
- Read-only grid (no accidental edits)
- Stable selection behavior
- View selected cell value in a popup
- Designed with clear separation between UI and logic

## Tech Stack
- C#
- .NET WinForms

## How to Run
1. Open the solution file in Visual Studio
2. Build the solution
3. Run the application
4. Click **Load CSV** and select a CSV file

## Notes
This project demonstrates practical handling of WinForms DataGridView
behavior and user interaction edge cases.
